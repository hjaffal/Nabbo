# Nabbo Documentation

> Don't remember it. Nabbo it.

Nabbo is the command center for modern family life. It captures messy household information from shared messages, forwarded emails, images, voice notes, documents, screenshots, and free text — then uses AI to extract what matters, detect changes, identify risks, assign responsibility, build preparation checklists, and show what needs to happen next.

## Quick Overview

| Aspect | Detail |
|--------|--------|
| **Target User** | The operating parent — the one who carries the household logistics load |
| **Core Behavior** | Capture → Understand → Verify → Assign → Execute → Learn |
| **Input Methods** | Mobile share, email forwarding, free text, voice input |
| **Key Screens** | Home, Review Inbox, Today Command Center, Settings |
| **Positioning** | Family logistics assistant / household operations layer |
| **Business Model** | Freemium subscription |

## Principles

- Reduce mental load, don't organize it
- Convert information into action
- Show uncertainty instead of hiding it
- Make ownership visible
- Help the family prepare before stress starts
- Detect changes and risks early
- Avoid becoming another place to manage

Every feature must pass one test: **Does this help the family act earlier, prepare better, assign ownership, or avoid a miss?**
